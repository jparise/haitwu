Overview

Haitwu is fairly simple. It fetches the last 200 public tweets for the requested screen name and scans their text for sequences of words that match the 5-7-5 syllable haiku "formula".

To reduce load, each screen name's results are cached for one hour. New haiku-compliant tweets won't show up until the screen name's cache entry expires.

Unfortunately, timeline request errors occur fairly often due to Twitter's rate limiting. Haitwu uses the public (non-authenticated) Twitter API from shared Google AppEngine IP addresses, so it runs into the rate limit fairly often.

Due to the free-form text patterns often found in tweets, haiku detection isn't always completely accurate. Various bits of punctuation can throw things off, for example. Improvements will come in time.

Technology

Haitwu is written in Python on top of Google AppEngine.

Haiku detection is based on Jonathan Feinberg's haikufinder module, which uses the Natural Language Toolkit.
